💬 Conectar com WhatsApp

Mandar mensagem no WhatsApp e o gasto cair direto aqui exige um servidor conectado à API oficial do WhatsApp (Meta) e uma conta na nuvem para guardar os dados — esse site sozinho (arquivo estático no GitHub Pages) não consegue receber mensagens.

Isso é possível de construir como um projeto à parte: um bot de WhatsApp (via Meta Cloud API ou Twilio) que recebe sua mensagem, ex. "gastei 30 no mercado", e grava num banco de dados (Firebase ou Supabase). Este site então mudaria para buscar os dados desse banco em vez do armazenamento local do navegador.
