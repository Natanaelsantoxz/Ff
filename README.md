# Minhas Finanças

Site simples de organização financeira pessoal (contas a pagar, metas de poupança, renda e calculadora de investimento).

## Como publicar no GitHub Pages

1. Crie um repositório novo no GitHub (ex: `minhas-financas`).
2. Envie os arquivos deste ZIP para o repositório (pode arrastar e soltar pela interface do GitHub em "Add file > Upload files").
3. Vá em **Settings > Pages**.
4. Em "Branch", selecione `main` e a pasta `/ (root)`, depois clique em **Save**.
5. Espere 1–2 minutos. O GitHub vai te dar um link do tipo:
   `https://seu-usuario.github.io/minhas-financas/`

## Como atualizar depois

Sempre que quiser mudar algo, é só substituir o arquivo `index.html` no repositório (upload de novo arquivo com o mesmo nome) e o site atualiza sozinho em alguns minutos.

## Sobre os dados salvos

Este site guarda as informações no navegador de cada aparelho (localStorage), não em uma conta de usuário na nuvem. Ou seja, os dados não se sincronizam entre celular e computador, por exemplo — cada aparelho tem seu próprio "banco de dados" local.
